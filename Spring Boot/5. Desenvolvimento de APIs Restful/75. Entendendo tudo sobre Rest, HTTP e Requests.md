# Modelagem de Contratos de API’s

# Contrato de API?

Um contrato de API é uma especificação que define todas as regras e diretrizes para a interação entre um cliente e uma API. Ele funciona como um acordo formal entre o provedor da API e os consumidores, garantindo que todos saibam exatamente como devem se comunicar.

# Definindo um Contrato

- [ ] Identificação do Recurso
- [ ] Método/Verbo HTTP apropriado
- [ ] Código de Resposta apropriado
- [ ] Definição do Payload de Request e Response
- [ ] Definição dos Headers

# Identificação de Recursos

# Utilização de Substantivos no plural no lugar de verbos

O verbo da ação é o Método HTTP!

|**Operação**|**Incorreto**|**Correto**|
|---|---|---|
|Cadastrar Cliente|/cadastro-cliente/salvar|/clientes|
|Atualizar Cliente|/atualizar-cliente|/clientes/{id}|
|Dados do Cliente|/dados-cliente/obter|/clientes/{id}|
|Excluir Cliente|/excluir/cliente|/clientes/{id}|

# Método HTTP Apropriado

O verbo da ação é o Método HTTP!

|**Operação**|**Incorreto**|**Correto**|
|---|---|---|
|Cadastrar Cliente|GET /cadastro-cliente/salvar?nome=fulano&cpf=123456789|POST /clientes|
|Atualizar Cliente|PUT /atualizar-cliente|PUT /clientes/{id}|
|Dados do Cliente|POST /dados-cliente/obter/1|GET /clientes/{id}|
|Excluir Cliente|POST /excluir/cliente?id=1|DELETE /clientes/{id}|
|Pesquisa Clientes|GET /pesquisa-clientes/nome/fulano|GET /clientes?nome=fulano&ativo=true|

# Estrutura Hierárquica (subrecursos)

|POST /usuario/{id}/pedidos|

# Código de Resposta apropriado

Representa na resposta o Status do que aconteceu para aquela requisição

|**Operação**|**Request**|**Códigos Sucesso**|**Códigos de Erro**|
|---|---|---|---|
|Cadastrar Cliente|POST /clientes|201, 200|400, 409, 422|
|Atualizar Cliente|PUT /clientes/{id}|204, 200|400, 409, 422|
|Dados do Cliente|GET /clientes/{id}|200|404|
|Excluir Cliente|DELETE /clientes/{id}|204, 200|404, 400, 409|
|Pesquisa Clientes|GET /clientes?nome=fulano&ativo=true|200|Não tem!|
|Autenticação|POST /auth|200|401|
|Erro não tratado|Qualquer Request|Não Tem!|500|
