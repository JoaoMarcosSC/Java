
## Anotação `@Entity`

### O que é

`@Entity` indica que essa classe é uma **entidade JPA** e deve ser gerenciada pelo Hibernate/JPA.
- Sem essa anotação, o Hibernate **ignora a classe**
- É obrigatória para persistência no banco

## Anotação `@Table`
### O que é
Define **explicitamente** o nome da tabela e o schema no banco de dados.
- `name = "autor"` → nome da tabela
- `schema = "public"` → schema do PostgreSQL

Se não fosse informada:
- O nome da tabela seria inferido automaticamente
- O schema padrão seria utilizado

### Trecho do código
```Java
@Table(name = "autor", schema = "public")
```


## Lombok: `@Getter` e `@Setter`

### O que é
Essas anotações fazem o Lombok gerar automaticamente:
- Métodos `getX()`
- Métodos `setX()`

Isso evita escrever código repetitivo.

### Importante
- Os métodos **existem em tempo de compilação**
- Você não os vê no código, mas o Java enxerga

### Trecho do código
```Java
@Getter 
@Setter
```

## Identificador da entidade – `@Id`

### O que é
Marca o atributo que será a **chave primária** da tabela.
- Cada entidade JPA deve ter **exatamente um `@Id`**
- Esse campo identifica unicamente o registro

## Geração automática do ID – `@GeneratedValue`

### O que é

Define **como o valor do ID será gerado**.
- `GenerationType.UUID`:
    - O próprio Hibernate gera um UUID
    - Não depende de sequence ou auto increment do banco
    - Ideal para sistemas distribuídos

### Trecho do código
```Java
@GeneratedValue(strategy = GenerationType.UUID)
```

## Mapeamento da coluna `id`

### O que é
`@Column` define o mapeamento do atributo para uma coluna da tabela.
- `name = "id"` → nome da coluna no banco
- Tipo Java: `UUID`
- Tipo no banco: UUID (PostgreSQL)

### Trecho do código
```Java
@Column(name = "id") 
private UUID id;
```

## Coluna `nome`

### O que é

Define a coluna `nome` com regras de validação no banco.
- `length = 100` → tamanho máximo
- `nullable = false` → não pode ser `NULL`

Essas regras influenciam:
- Geração de DDL
- Validações de persistência

### Trecho do código
```Java
@Column(name = "nome", length = 100, nullable = false) 
private String nome;
```


## Coluna `data_nascimento`
### O que é
Mapeia uma data sem horário (`LocalDate`) para o banco.
- Ideal para datas como nascimento
- Não armazena timezone nem hora

### Trecho do código

```Java
@Column(name = "data_nascimento", nullable = false) 
private LocalDate dataNascimento;
```

## Coluna `nacionalidade`

### O que é
Define a nacionalidade do autor com limite de tamanho.
- `length = 50`
- `nullable = false`

### Trecho do código

```Java
@Column(name = "nacionalidade", length = 50, nullable = false) 
private String nacionalidade;
```
